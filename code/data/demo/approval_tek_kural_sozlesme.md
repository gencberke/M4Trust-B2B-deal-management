# MAL ALIM SÖZLEŞMESİ (Onay-odaklı demo)

Bu sözleşme, M4Trust demo/manuel test amaçlıdır. Yukarıdaki
`[[m4trust-fake-profile: approval]]` marker'ı, `FakeExtractionService`'in
**onay profili** fixture'ını (tek `approval` tetikli kural) açıkça seçer. Marker
yokken de varsayılan profil `approval`'dır (mevcut davranış bit-bit korunur).

## Taraflar

- **Alıcı:** Örnek Alıcı A.Ş.
- **Satıcı:** Örnek Satıcı Ltd. Şti.

## Ticari Şartlar

- Konu: Endüstriyel Pompa — 10 adet
- Toplam tutar: 100.000 TRY
- Teslim tarihi: 2026-09-01

## Ödeme Kuralları

1. **Sipariş onayı:** Tarafların onayıyla tutarın **tamamı (%100)** ödenir.
